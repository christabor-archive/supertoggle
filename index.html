jQuery supertoggle

Tired of writing boilerplate toggle() code in jquery?
Do it powerfully, declaratively, and easily instead.


Add data-toggle="swappable" to your elements, and describe the scenario you'd like to see in JSON, right in the same place. For example <a href="#" data-swap='{"#left.cols": [{".from": ".to"}]}'>. It just works!

Usage

Initialize

  1. Add the script to your document, after jQuery.
  2. Initialize whatever selector you want. E.g. $('[data-toggle="swappable"]').on('click', supertoggle);

Declare

Elements must have data-toggle="swappable" set, and declarations are done via data-swap='{"#selector": [{".from1": ".to1"}, {".from2": ".to2"}]}' Where each key is a jquery selector, and each value is an array of objects with one key and one value, describing the swap from, and swap to values. There's no limit to how many you can use.

More advanced usage

See the above examples for multiple selectors, pseudo-selectors and more.
